# خريطة الطريق — Deen Roadmap

منصة تعليمية تفاعلية لتأسيس المسلم في طريق طلب العلم الشرعي وتزكية النفس، مبنية على منهج متدرج ومنظم.

## المحتوى

- **المستوى الأول** — تمهيد طلب العلم (مفاتيح الطلب، مهارات التعلم، مداخل العلوم) والتزكية بمرحلتيها (تأسيس وإصلاح، بناء وترقّي)
- **300+ درس** مرئي من يوتيوب مع تنبيهات صوتية قبل كل قسم
- تصميم عربي متجاوب يدعم الهواتف والأجهزة المختلفة

## التقنيات

- React 19 + Vite 8
- React Router DOM
- CSS مخصص (بدون مكتبات خارجية)
- نشر على Vercel

## التشغيل المحلي

```bash
npm install
npm run dev
```

## البناء

```bash
npm run build
npm run preview
```

## النشر

المشروع مُهيأ للنشر على [Vercel](https://vercel.com) مباشرة — اربط المستودع وسيتم البناء والنشر تلقائيًا.

## الرخصة

هذا المشروع لوجه الله تعالى. يُسمح بالنسخ والنشر والتعديل بشرط عدم التحريف والإشارة للمصدر.
